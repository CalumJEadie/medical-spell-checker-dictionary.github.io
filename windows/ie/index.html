Internet Explorer

Use an automatic installer

  1. Download medical_spell_checking.exe.

  2. Click "run this program."

  3. Try mis-spelling "adenomyosis" in a web form.

Manual installation

  1. Download the file default.dic.

  2. Open Windows Explorer.

  3. Type %appdata%\Microsoft\Spelling\en-GB\ in the location bar at the top of Windows Explorer.

  4. Drag the file default.dic you downloaded into this folder.

  5. Click the button "yes I do want to replace this file".

  6. Try mis-spelling "adenomyosis" in a web form.
